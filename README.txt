===========
localCIDER
===========

Version 0.1.0


ABOUT
========
localCIDER is both a Python library and a set of command line tools for 
computing many of the parameters associated with intrinsically disodered 
proteins (IDPs) which can be calculated by the CIDER webserver.

For more information please visit `the localCIDER webpage <http://pappulab.github.io/LocalCIDER/>`_.
